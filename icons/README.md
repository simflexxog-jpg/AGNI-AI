Place Lottie JSON animation files here named to match the icon keys used by the app.

Expected filenames (examples):
- settings.json
- menu.json
- more_vert.json
- mic.json
- image.json
- attach_file.json
- send.json

Recommended sources:
- Iconscout Lottie packages (download the JSON and save under the names above)
- LottieFiles (download JSON and copy here)

After adding the JSON files, reload the app; icons will automatically be replaced by `lottie-player` instances and will play on hover/click.
